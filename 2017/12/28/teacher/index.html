绵阳平武一段经历

发表于 2017-12-28 | 分类于 生活随记


2012年7月-8月绵阳平武

想要写一段回忆，竟然不知如何开头，不知如何让读者看起来不那么乏味。又想了想，能来我的站溜达的（非技术类），多半可能是比较关心我的…这样一来，我也就能写的比较随意了。记录点滴，一方面是为了让关心者了解我的动态，另一方面也是为记录过去吧（防止老年痴呆了失忆…），记录的过程也是一个人自我对话的过程，不断的为自己明确方向,let’s go。

记得第一年去成都上学，从家到学校，火车车门刚打开的瞬间就被外面的热气震到了，之前也听说闷热，但是实际的感觉远比想象的强烈。从此就开始了自己四年的大学生活，不对，少了点东西，“四年的大学生活”前面需要个修饰词，那就用“苦逼”吧，苦逼是别人认为的，而我的感觉就是享受和幸福。这里跑的有点远，以后再更新学校期间的点滴吧，这次我主要记录的是大一暑假期间的“支教生活”

大一快要临近暑假的时候，学校就陆陆续续来了很多组织支教活动的团体，当时觉的暑假放假时间长，而且这种活动也很有意义，于是就决定报名参加，当时我认识的同学里，也是有好几个参与了这次活动。其实这次活动不能算是严格意义上的公益支教吧，校方是向学生收取了我认为不太合理的费用的，这里就不多讲了…（省略的部分自有发挥吧）。但是吧，不用支教这个词，又不好描述，下文中就暂且继续延用“支教”吧。

于是乎，暑假如期而至，我们也开始了自己的支教生活，对了，还有一段插曲，就是支教的老师是需要筛选的，先是报名，然后简单试讲，再进行自我介绍就OK了，反正都是“吹牛”。后来我被分到了绵阳平武这个地方，当时听说这个地方是要去七个老师的。很快我们就去了绵阳平武的一所小学，35天的支教生活就此开始。

先放一张集体合影图，对我们几个老师有个直观体验，我是打马呢还是不打呢，算了还是不打了，反正看着比现在帅 :)。

七个老师当然都有各自的分工了，图片中打个小领带的是我们的负责人，主要负责我们不会被饿死…讲真他确实也很有担当，负责我们的开支、场地安排、和当地的一些人事沟通等等，毕竟总要有个负责人嘛。你没看错，就是还有三个妹子，有个川师的，有两个是西华的，每个妹子都很有特点，大家相处的都很愉快，她们主要负责小学的课程，具体是哪门课，也记不太清了,当时她们还被顽皮的小朋友气的不行，哈哈。另外两个男生，一个是数学老师，一个是语文老师。下面再来张可爱小朋友的合影照（有木有托儿所的赶脚）

这张是下午放学后，给小朋友们放个电影。当时手机像素比较渣，就忍忍吧。

这张是一个老师认真上课时的样子

下面就到我了，我带的是高一的和初二升初三的物理化学。每天4个小时工作量，上下午各两个小时。讲课期间发现资料有点少，然后就去网吧找资料，筛选完后就去打印出来，让他们练习，每天就需要稍微的准备一下，不过也很轻松。记得12年7月刚好有伦敦奥运会，那天我就在网吧看了一晚上的开幕式（可能当时晚上闲的*疼才去的），早上就稍微眯了一会，白天也不觉的累。我们住一个幼儿园里，睡的床很小，夏天天气热还好能将就下….

平时的饭菜是我们几个轮流着做的，谁下午没课就早点去厨房做饭，我们一起的几个妹子都很会做饭，当然做的味道也很好了，一般我做饭比较少，是不是很爽…不过我洗碗多，是不是很尴尬。菜也是自己买的，印象里我买菜的次数比较少，我们一起川师的妹子每天真的很勤快，做了很多事情。来几张厨房做饭的照片，都是忙碌的背影，很赞。

其实每天吧，基本都差不多，我们偶尔也会去附近的地方散散步，就像当地的县城有条河，河边的道路修建的很好，每天晚些时候都会有很多人去散步。河很大，水很湍急，太阳刚要落山，阵阵凉风非常惬意。

三十多天也是很快，每天非常规律，月末的时候给班级里表现比较好的学生颁发了奖状，他们都很开心，毕竟小孩子都是需要鼓励的嘛。最后一天大家收拾好了行李，准备出发了，就这样基本上就结束了支教生活。

开始迎来了新的大二生活。

生活就像拼积木，只要努力去做，没有拼成之前，没有人知道会是什么样子的。尽力去做就好。
